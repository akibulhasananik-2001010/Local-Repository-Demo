# Local Repository Demo

A simple static website created to demonstrate working with a local Git repository.

## Overview

The website consists of a basic HTML page styled with CSS and serves as an introduction to version control, repository management, and how to push files by using vs code terminal.

## Features

* Simple and lightweight web page
* Beginner-friendly project structure
* Demonstrates local Git repository workflow

## Technologies Used

* HTML5
* CSS3
* Git

## Project Structure

```text
Local-Repository-Demo/
│
├── index.html      # Main webpage containing the website structure and content
├── style.css       # Stylesheet responsible for the website's appearance and layout
└── README.md       # Project documentation and setup instructions
```

## How to Run Locally

1. Clone the repository.
2. Open the project folder.
3. Open the `index.html` file in any web browser.

## Learning Purpose

This repository was created for learning purposes to understand:

* Local Git repository creation
* Git version control workflow
* Connecting local repositories to GitHub
